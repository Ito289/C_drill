\subsection*{問1}

次の虫食い算を解くプログラムを（二重ループと{\ttfamily if}
と剰余を用いて）書け。
\begin{verbatim}
　１□□
×　１□
────
□□□１
□□□
────
□１１□
\end{verbatim}
つまり、$100\times10$から$199\times19$のかけ算{\ttfamily a*b}の
二重ループを考え、{\ttfamily if}文で判定し、
筆算の計算結果を{\ttfamily printf}で表示する。


\ifnum \anss=1
\begin{verbatim}

\end{verbatim}
\vspace{15cm}
\vfill
\mbox{}\\
\else
\begin{verbatim}
int a = 100, b;
while(a <= 199){
  b = 10;
  while(b <= 19){
    if( ((b-10)*a) % 10 == 1 &&
        1000 <= ((b-10)*a) && ((b-10)*a) < 10000 &&
        ((a*b)/10) % 100 == 11 &&
        1000 <= b*a && b*a < 10000
        ){

      printf(" %d\n", a);
      printf("x %d\n", b);
      printf("----\n");
      printf("%d\n", (b-10)*a);
      printf("%d\n",  a);
      printf("----\n");
      printf("%d\n",  a*b);
    }
    b = b + 1;
  }
  a = a + 1;
}


正解  まちがい
 183   101
x 17  x 11
----  ----
1281   101
183   101
----  ----
3111  1111
\end{verbatim}
\vspace{4cm}
\mbox{}
\fi
