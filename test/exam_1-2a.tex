\subsection*{問1}

以下の代入文が上から順番に実行されるとき、全ての代入文の
実行が終わった時点での各変数の値を書け。

\begin{verbatim}
y = 1;
x = 2;
z = sqrt(9);
x = x + 14;
y = x / 2;
x = x + 4;
y = z = z+x * 4;
z = (z - 3) / 40*2;
\end{verbatim}


{\ttfamily x}の値:\an{\quad 20}\\

{\ttfamily y}の値:\an{\quad 83}\\

{\ttfamily z}の値:\an{\quad 4}\\
